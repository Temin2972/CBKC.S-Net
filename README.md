# S-Net - Nền tảng website hỗ trợ tâm lý học đường
#### From team CBKC
## Tính năng
- Chat riêng tư giữa học sinh và các giáo viên tư vấn.
- Trang cộng đồng ẩn danh hỗ trợ đăng bài và bình luận.
- Hệ thống AI kiểm duyệt nội dung tự động trước khi đựợc đăng lên trang cộng đồng, nếu hệ thống AI không hoạt động hoặc nội dung khó phân loại, nội dung sẽ được để cho các giáo viên tư vấn xét duyệt.
## Nền tảng
- **Frontend:** React 18 + Vite
- **Backend:** Supabase (PostgreSQL + Real-time + Auth + Storage)
- **AI system:** AI API bất kỳ
- **Styling:** Tailwind CSS
- **Icons:** Lucide React
- **Deployment / Hosting:** Vercel
## Triển khai
Bất kỳ ai cũng có thể sử dụng nền tảng nèy để triển khai 1 trang web riêng của họ. Xem thêm tại [DELOYMENT.md](DEPLOYMENT.md).
